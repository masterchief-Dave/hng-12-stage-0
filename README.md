## 📌 Description

## 🛠️ Project Setup
To install dependencies, run:

```bash
pnpm install
```

---

## 🚀 Running the Project

### Development Mode
```bash
pnpm run start:dev
```

### Production Mode
```bash
pnpm run start:prod
```
